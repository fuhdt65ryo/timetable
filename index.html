▶ 初回セットアップ手順
  1. Googleスプレッドシートで新しいシートを作成する
  2. メニューの「拡張機能」→「Apps Script」を開く
  3. 表示されたコードをすべて削除し、下のコードを貼り付ける
  4. 「保存」してから「デプロイ」→「新しいデプロイ」をクリック
  5. 種類:「ウェブアプリ」、アクセス:「全員」に設定して「デプロイ」
  6. 表示されたウェブアプリURLをコピーして下の欄に貼り付ける
function doGet() {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  return ContentService.createTextOutput(sheet.getRange('A1').getValue() || '{}')
    .setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.getRange('A1').setValue(e.postData.contents);
  return ContentService.createTextOutput('{"ok":true}')
    .setMimeType(ContentService.MimeType.JSON);
}
